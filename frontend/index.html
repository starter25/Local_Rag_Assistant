사용 설명서

현재 앱에서 가장 자주 쓰는 흐름만 정리했어.

1. 문서 기반 답변

문서를 업로드한 뒤 문서 기반 모드로 질문하면 문서에서 찾은 내용 안에서만 답합니다.

2. 문서+해석 답변

문서 내용을 기준으로 답하되, 추가 해석이나 조언이 필요할 때 사용합니다.

3. 일반 AI

문서 없이 일반 질문, 요약 아이디어, 코딩 질문을 할 때 사용합니다.

4. 대화 기억

같은 채팅 안에서는 이전 대화 맥락을 이어서 답합니다. 필요하면 왼쪽 채팅 목록에서 새 채팅을 시작하세요.
